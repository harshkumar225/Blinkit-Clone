# Blinkit-Clone

A responsive Blinkit Clone website built using HTML, CSS, and JavaScript.  
This project recreates the user interface of the Blinkit grocery delivery platform and helps practice modern front-end web development skills.

## 🌐 Live Demo
https://harsh-blinkit-clone.netlify.app/

## ✨ Features
- Responsive design for mobile and desktop
- Product grid layout
- Navigation bar with search
- Clean and modern UI inspired by Blinkit

- HTML
- CSS
- JavaScript
